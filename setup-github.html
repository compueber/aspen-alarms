🚀 Setup GitHub ASPEN PWA

3 métodos para ter seu repositório funcionando em minutos!

🥇 Método 1: Fork do Template (30 segundos)

Use um dos repositórios template já prontos:

1. Acesse um destes templates:
📦 Microsoft PWA Template
🔧 Google PWA Starter
⚡ PWA Builder Template
2. Clique "Use this template" → "Create repository"
Nome: aspen-alarms
Descrição: Sistema de alarmes ASPEN Digital PWA
3. Substitua os arquivos pelos nossos artefatos
Edite cada arquivo diretamente no GitHub com o conteúdo dos artefatos

🥈 Método 2: Criar do Zero (2 minutos)

1. Crie novo repositório
github.com/new
Nome: aspen-alarms
Descrição: 🔔 Sistema de alarmes inteligente ASPEN Digital PWA
✅ Public   ✅ Add README
2. Adicione arquivos um por um:
aspen-alarms/ ├── index.html ← Artefato "alarm_system" ├── manifest.json ← Artefato "manifest_json" ├── sw.js ← Artefato "service_worker" ├── css/ │ └── styles.css ← Artefato "styles_css" ├── js/ │ └── app.js ← Artefato "app_js" ├── generate-icons.html ← Artefato "icon_generator" └── README.md ← Artefato "readme_md"
3. Para cada arquivo:
• Add file → Create new file
• Digite nome (ex: index.html)
• Cole conteúdo do artefato
• Commit new file

🥉 Método 3: Upload em Lote (1 minuto)

1. Baixe todos os arquivos localmente
Salve cada artefato como arquivo separado na estrutura correta
2. Upload da pasta completa
No GitHub: Add file → Upload files
Arraste a pasta aspen-alarms/ completa

📡 Ativar GitHub Pages (Qualquer método)

1. No seu repositório:
Settings → Pages
2. Configure:
Source: Deploy from a branch
Branch: main
Folder: / (root)
3. Aguarde 2-5 minutos
URL final: https://seuusuario.github.io/aspen-alarms

📱 Gerar APK (Opcional)

1. Acesse PWA Builder:
pwabuilder.com
2. Cole URL do GitHub Pages:
https://seuusuario.github.io/aspen-alarms
3. Package → Android → Download APK
APK pronto para instalar! 🎉
